# fieldbook-relase

Release artifacts for Fieldbook plugin.

## Install via bash (no zip)

```bash
curl -fsSL https://raw.githubusercontent.com/Prashanth-BC/fieldbook-relase/main/install.sh | bash -s -- "/path/to/your/vault"
```
